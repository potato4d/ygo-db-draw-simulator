このツールについて

遊戯王公式データベースに『おためし5枚ドロー』の機能を追加できる拡張機能です。

確率計算などを行っていても、実際にドローしてみないと体感値や細かな動きはわからないもの。ドローのシミュレーションをともなう調整のおともにぜひ。

インストール方法

Chrome Web Store にてインストールできます。

上記のリンクよりアクセスし、『Chrome へ追加』より導入してください。インストールが完了したら、自動で拡張機能が有効化されます。

つかいかた

拡張機能の導入後、遊戯王公式データベースのデッキ表示ページへとアクセスすると、サイドデッキの下に『おためし5枚ドロー』が追加されています。

アクセスした時点でメインデッキからランダムに5枚ピックされていますが、必要に応じて『5枚ドロー』による再抽選や、『+1枚引く』をご活用ください。
